# Images Folder

This folder contains static image assets for the portfolio site.

## Folder Structure

- `portfolio/` - Portfolio project images
- Add other folders as needed (e.g., `blog/`, `icons/`, etc.)

## How to Add Images

1. Place your image files in the appropriate subfolder
2. Reference them in your code using the path: `/images/folder/filename.ext`

## Example

If you add `nuclear-doc.jpg` to `portfolio/`, reference it as:
```
image: '/images/portfolio/nuclear-doc.jpg'
```

## Recommended Image Formats

- **JPEG** (.jpg) - Photos, screenshots
- **PNG** (.png) - Images with transparency, logos
- **WebP** (.webp) - Modern format, best compression
- **SVG** (.svg) - Vector graphics, icons

## Image Size Recommendations

- Portfolio thumbnails: 600x600px or 800x800px
- Blog covers: 1200x630px
- Keep file sizes under 500KB when possible
